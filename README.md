FarmCon is an innovative application that bridges the gap between consumers, farmers, and agricultural business professionals by providing comprehensive information about farm products and their production locations. This repository houses the project's source code and documentation.

## Description

FarmCon facilitates the acquisition of farm products for both business and consumption purposes. Users can access detailed information about specific farm products, including their location, quantity, price, duration, and availability.

## MVP Features

- **Landing Page**: Provides essential information about the product, services offered, contact details, and the team.
- **Authentication & Authorization**: Secure signup and login functionalities with role-based access control.
- **Dashboard**:
  - **Super Admin**: Full system access with CRUD operations on all entities.
  - **Admin**: User and product management capabilities.
  - **Users**: CRUD features for managing personal products.
- **Product Management**:
  - Comprehensive product information including name, description, planting/harvesting periods, and production locations.
- **User Interaction**:
  - Detailed product pages, interactive maps, and feedback/rating system.
- **Farmers' Upload Feature**:
  - Farmers can upload product details and images for display.

## Learning Objectives

- Prototype development using React.js and Bootstrap.
- Consumption of data via endpoints.
- Utilization of React Hooks (e.g., useState, useEffect), Redux, and other related technologies.

## Technologies Used

- **Front-end**: React, CSS3, Bootstrap
- **Back-end**: Python, MySQL/SQLite
- **Methodology**: RESTful API

## Third-Party Services

- **Google Maps Integration**: Provides location-based services and enhances user experience with interactive maps.
- **Weather API**: Integrates weather data for agricultural insights and planning.

## Challenges Identified

- **Data Management and Scalability**: Handling large volumes of data efficiently.
- **User Authentication and Authorization**: Ensuring secure access and data privacy.
- **Google Maps Integration**: Implementing location-based features accurately.
- **Real-time Data Updates**: Synchronizing backend data with frontend displays.
- **User Experience Design**: Designing an intuitive interface for diverse users.
- **Testing and Quality Assurance**: Comprehensive testing to ensure reliability and performance.
